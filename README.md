# AI Media Launcher 설치 매뉴얼

이 문서는 AI Media Launcher를 설치하고 실행하는 방법을 단계별로 설명합니다.

## 1. Node.js 설치

이 프로그램은 Node.js 환경에서 실행됩니다. 먼저 Node.js를 설치해야 합니다.

1.  아래 링크를 클릭하여 Node.js 다운로드 페이지로 이동합니다.
    *   [https://nodejs.org/ko/download](https://nodejs.org/ko/download)
2.  자신의 운영체제(Windows, macOS 등)에 맞는 버전을 다운로드합니다.
    *   일반적으로 "LTS (Long Term Support)" 버전을 권장합니다.
3.  다운로드한 설치 파일을 실행하고, 화면의 지시에 따라 설치를 완료합니다.

## 2. 소스 코드 다운로드

GitHub에서 프로그램의 소스 코드를 다운로드합니다.

1.  아래 링크를 클릭하여 프로젝트 저장소로 이동합니다.
    *   [https://github.com/gyette-ai/AI-media-launcher](https://github.com/gyette-ai/AI-media-launcher)
2.  페이지 우측 상단의 초록색 **Code** 버튼을 클릭합니다.
3.  **Download ZIP**을 클릭하여 소스 코드를 압축 파일로 다운로드합니다.
4.  다운로드한 ZIP 파일의 압축을 원하는 위치에 풉니다.

## 3. 터미널 열기 및 명령어 입력

이제 터미널을 통해 프로그램을 설치하고 실행합니다.

### 3.1 터미널 열기 (Windows 기준)

1.  압축을 푼 폴더(소스 코드가 있는 폴더)로 이동합니다.
2.  폴더 빈 공간에 `Shift` 키를 누른 상태로 마우스 오른쪽 버튼을 클릭합니다.
3.  메뉴에서 **"PowerShell 창 여기서 열기"** 또는 **"터미널에서 열기"**를 선택합니다.
    *   또는, 폴더 주소창에 `cmd`라고 입력하고 엔터를 치면 해당 경로에서 명령 프롬프트가 열립니다.

### 3.2 프로그램 설치 및 실행

터미널(또는 명령 프롬프트) 창이 열리면 아래 명령어들을 순서대로 입력하고 엔터(Enter)를 누르세요.

**1단계: 필요한 도구 설치하기**

```bash
npm install
```

*   이 명령어를 입력하면 프로그램 실행에 필요한 파일들을 자동으로 다운로드합니다.
*   완료될 때까지 잠시 기다려주세요.

**2단계: 프로그램 빌드하기 (실행 파일 생성)**

```bash
npm run build
```

*   설치가 완료되면 이 명령어를 입력하여 실행 파일을 생성합니다.
*   빌드가 완료되면 `release` 폴더 안에 설치 파일(예: `AI Media Launcher Setup 1.0.0.exe`)이 생성됩니다.
*   생성된 설치 파일을 실행하여 프로그램을 설치하고 사용할 수 있습니다.

---

**참고:**
*   `npm` 명령어가 실행되지 않는다면 Node.js가 제대로 설치되지 않았을 수 있습니다. Node.js 설치 후 컴퓨터를 재부팅하고 다시 시도해보세요.
