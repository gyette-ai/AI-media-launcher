# Launch Deck 설치 매뉴얼

이 문서는 **Launch Deck**을 설치하고 실행하는 방법을 단계별로 안내합니다.

> **⚠️ 주의 (Important): 이 프로그램은 Windows 운영체제 전용입니다.**
> (This application is compatible with Windows OS only.)

> **현재 버전:** v1.0.0
>
> [📝 변경 기록 (Changelog) 확인하기](CHANGELOG.md)

## 1. 프로그램 다운로드

1.  GitHub 저장소로 이동합니다.
    *   [https://github.com/gyette-ai/AI-media-launcher](https://github.com/gyette-ai/AI-media-launcher)
2.  우측 상단의 **Code** 버튼을 클릭합니다.
3.  **Download ZIP**을 선택합니다.
4.  ZIP 파일의 압축을 원하는 위치에 해제합니다.
5.  폴더 내의 `Launch Deck Setup 1.0.0.exe` 파일을 실행하여 설치를 진행합니다.

## 2. 프로그램 설치 및 실행

### 2.1 Windows에서 실행하기

1.  다운로드받은 `Launch Deck Setup 1.0.0.exe` 파일을 실행합니다.
2.  자동으로 설치가 진행되며, 설치가 완료되면 프로그램이 실행됩니다.
3.  바탕화면에 생성된 **Launch Deck** 아이콘을 통해 언제든지 프로그램을 다시 실행할 수 있습니다.

> **Note:** "Windows의 PC 보호" 경고창이 뜰 경우:
> 1. **"추가 정보"**를 클릭합니다.
> 2. **"실행"** 버튼을 클릭하여 설치를 계속합니다.

## 3. 참고 사항

*   이 프로그램은 Windows 환경에 최적화되어 있습니다.
*   자동 업데이트 기능이 포함되어 있어, 새로운 버전이 나오면 알림을 받을 수 있습니다.
